🚀 How to Embed in Webflow:

  1. Create your destination page in Webflow (e.g., "/dog-names" or "/premium-names")
  2. Add an HTML Embed element where you want the widget
  3. Paste the code below and customize the data-cta-url
  4. Replace YOUR_GEMINI_API_KEY with your real API key
  5. Publish your site!

📋 Webflow HTML Embed Code:

<!-- Unified Dog Name Widget for Webflow -->
<div 
  class="dog-name-unified-widget" 
  data-api-key="YOUR_GEMINI_API_KEY"
  data-cta-url="/premium-dog-names">
</div>

<script src="https://cdn.jsdelivr.net/gh/zaynnzhong/dog-name-generator-widget@main/dist/dog-name-unified-widget.iife.js"></script>
<script>
  window.addEventListener('load', function() {
    if (window.renderDogNameUnifiedWidget) {
      window.renderDogNameUnifiedWidget();
    }
  });
</script>
